# Pullrequests Service

Сервис назначения ревьюверов на Pull Request’ы, управления командами и пользователями.

## Загрузка и запуск

Скачать проект:

```bash
git clone https://github.com/ilvladik/pullrequests.git
cd pullrequests
docker-compose up --build
```
Сервис будет доступен на http://localhost:8080.

## Исправление проблемы с пользователями

При создании команды можно было указать пользователя с ID, который уже существует в другой команде. Это приводило к ошибке в момент создания команды.

### Решение

Добавлена доменная ошибка `USER_EXISTS`. При создании команды проверяется, не существует ли пользователь уже в другой команде.
В случае конфликта возвращается ошибка с кодом `USER_EXISTS`
