# Pullrequests Service

Сервис назначения ревьюверов на Pull Request’ы, управления командами и пользователями.

## Загрузка и запуск

Скачать проект:

```bash
git clone https://github.com/ilvladik/pullrequests.git
cd pullrequests
docker-compose up --build
```
Сервис будет доступен на http://localhost:8080.
